# Save the Date - Invitación Interactiva

## Descripción

Este proyecto es una invitación interactiva para eventos, desarrollada en HTML, CSS y JavaScript. Está diseñada para ofrecer una experiencia visual atractiva y moderna, permitiendo a los usuarios interactuar con la invitación de manera dinámica.

## Características

- **Diseño responsivo**: Adaptado para dispositivos móviles y escritorio
- **Interactividad**: Elementos animados y transiciones suaves
- **Personalizable**: Fácil de adaptar para diferentes tipos de eventos

## Despliegue

Este proyecto está vinculado a **Netlify** para su despliegue continuo. Cada vez que se realizan cambios en el repositorio, Netlify automáticamente actualiza la versión en producción.

### Enlace de producción

El sitio está disponible en: [savethedate-neoproducciones.netlify.app](https://savethedate-neoproducciones.netlify.app/)

## Estructura del Proyecto

```
/
|-- index.html            # Archivo principal de la invitación
|-- README.md             # Documentación del proyecto
```

## Uso

1. Clona el repositorio
2. Modifica el archivo `index.html` con los datos de tu evento
3. Los cambios se desplegarán automáticamente en Netlify

- HTML5
- CSS3
